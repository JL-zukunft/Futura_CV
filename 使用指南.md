## 第一部分：认识项目（5 分钟）

### 1. 项目是什么？

这是一个**个人作品集网站**，可以用来展示您的设计案例、项目经验等，就像您的：
- 📄 **在线简历** - 展示您的专业背景
- 🖼️ **作品展览馆** - 展示您的项目案例
- 📊 **数据仪表盘** - 统计访客数据

**技术特点：**
- 💻 前端：HTML/CSS/JavaScript（响应式设计，支持手机/平板/电脑）
- 🔧 后端：腾讯云云开发
- 💾 数据库：腾讯云 MongoDB
- 💰 费用：完全免费（免费额度内）

---

### 2. 项目结构（像看文件夹一样简单）

```
📁 项目根目录 (e:\00jobs\self_html\)
│
├── 📄 portfolio.html                    ← 【主页面】网站的"门面"
│
├── 📁 frontend/                         ← 【前端文件夹】
│   └── api-client.js                   ← 【传话筒】帮前后端传递消息
│
├── 📁 backend-tencent/                  ← 【腾讯云后端】
│   ├── api/futura-html-api/index.js    ← 【云函数代码】
│   ├── scripts/init-database.js        ← 【数据库初始化】
│   ├── cloudbaserc.json                ← 【配置文件】
│   └── package.json                    ← 【依赖配置】
│
└── 📘 使用指南.md                       ← 【就是您现在看的这个】
```

**通俗理解：**
- `portfolio.html` = 餐厅的门面 + 装修 + 服务员
- `frontend/api-client.js` = 传话筒（服务员）
- `backend-tencent/` = 厨房（后台）
- 数据库 = 仓库（存储食材/数据）

## 第三部分：快速部署（15-20 分钟）

### 5. 方案 A：腾讯云部署（国内推荐）

**预计时间：** 15-20 分钟  
**适合人群：** 国内用户，需要数据统计功能

#### 第 1-3 分钟：注册与认证

1. **注册腾讯云**
   - 访问：https://cloud.tencent.com/
   - 点击 **免费注册**

2. **实名认证**（必须！）
   - 用户名 → 账号信息 → 实名认证
   - 准备身份证，完成人脸识别
   - ⏱️ 耗时：3-5 分钟

---

#### 第 4-6 分钟：开通云开发

1. **进入云开发控制台**
   - 访问：https://console.cloud.tencent.com/tcb
   - 点击 **开通云开发**

2. **创建环境**
   ```
   环境名称：portfolio
   区域选择：广州（推荐）
   计费模式：按量计费（免费版不收费）
   ```

3. **选择免费版**
   - ✅ 免费版：5GB 存储 + 500 万读写/月
   - ❌ 不要选付费版本

4. **保存环境 ID**
   - 格式：`portfolio-xxxxx`
   - 复制保存，后面要用

---

#### 第 7-9 分钟：创建数据库

1. **进入数据库管理**
   - 云开发控制台 → 数据库

2. **创建 9 个集合**
   ```
   cases          - 案例主表
   caseTabs       - 案例标签页
   caseImages     - 案例图片
   visitors       - 访客记录
   pageViews      - 页面浏览
   caseViews      - 案例浏览
   translations   - 多语言翻译
   siteConfig     - 系统配置
   aiGenerations  - AI 生成记录
   ```

3. **设置权限**
   - 公开内容（cases、caseTabs 等）：选择"读取全部数据不可修改数据"
   - 私密数据（visitors、pageViews 等）：选择"无权限"

---

#### 第 10-12 分钟：部署云函数

1. **安装工具**
   ```bash
   npm install -g cloudbase-cli
   ```

2. **修改配置**
   - 打开 `backend-tencent/cloudbaserc.json`
   - 将 `envId` 改为你的环境 ID

3. **登录并部署**
   ```bash
   cd backend-tencent
   npm install
   cloudbase auth:login
   cloudbase functions:deploy portfolio-api
   ```

4. **保存 API 地址**
   - 云开发控制台 → 云函数 → portfolio-api
   - 触发器 → 访问路径
   - 格式：`https://你的环境 ID.service.tcloudbase.com/api`

---

#### 第 13-15 分钟：部署前端

1. **修改 API URL**
   
   打开 `portfolio.html`，找到第 3828 行左右：
   ```javascript
   const API_BASE_URL = 'https://your-api-url.service.tcloudbase.com';
   ```

2. **部署到 GitHub Pages**
   ```bash
   cd e:\00jobs\self_html
   git init
   git add .
   git commit -m "Deploy portfolio"
   
   # 访问 https://github.com/new 创建仓库
   # 然后执行：
   git remote add origin https://github.com/你的用户名/portfolio.git
   git branch -M main
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 进入 GitHub 仓库 → Settings → Pages
   - Source 选择 main / /(root)
   - 保存

---

#### ✅ 验证部署

1. **访问网站**
   - 打开：`https://你的用户名.github.io/portfolio/`
   - 检查页面是否正常显示

2. **测试 API**
   - 打开浏览器开发者工具（F12）
   - Console 标签应该没有错误

3. **查看案例**
   - 点击案例卡片
   - 应该能正常展开和查看详情

## 第四部分：使用与维护（10 分钟）

### 7. 添加案例内容

#### 方法一：通过数据库控制台（推荐新手）

1. 登录云开发控制台
2. 进入数据库 → 选择集合（cases）
3. 点击 **添加记录**
4. 填入信息：
   ```json
   {
     "title": "我的案例标题",
     "subtitle": "案例副标题",
     "description": "案例描述",
     "cover_image": "/images/cover.jpg",
     "order_num": 1
   }
   ```
5. 保存

---

#### 方法二：通过 SQL 脚本（适合批量添加）

创建 SQL 文件，然后执行：

```sql
-- 添加案例
INSERT INTO cases (title, subtitle, description, cover_image, order_num)
VALUES 
  ('案例 1', '副标题 1', '描述 1', '/images/case1.jpg', 1),
  ('案例 2', '副标题 2', '描述 2', '/images/case2.jpg', 2);

-- 添加案例详情
INSERT INTO case_tabs (case_id, tab_name, tab_label, content)
VALUES 
  (1, 'overview', '概览', '<h3>项目背景</h3><p>...</p>'),
  (1, 'framework', '框架', '<h3>技术架构</h3><p>...</p>'),
  (1, 'solution', '方案', '<h3>实施方案</h3><p>...</p>');
```

### 9. 常见问题

#### Q1: 部署后看不到案例数据？

**答案：** 检查以下几点：
1. API URL 是否正确修改
2. 数据库是否创建了对应的集合
3. 是否插入了初始数据
4. 打开浏览器 F12，查看 Console 是否有错误

---

#### Q2: 云函数部署失败？

**答案：** 常见原因：
1. 没有安装依赖：`npm install`
2. 配置文件中的环境 ID 错误
3. 没有登录：`cloudbase auth:login`
4. 查看错误日志，根据提示修复

---

#### Q3: 如何修改网站的标题和描述？

**答案：** 打开 `portfolio.html`，找到：
```html
<title>Futura | 消费品 × AI</title>
<meta name="description" content="消费品×AI 产品设计师">
```
修改为你想要的文字。

---

#### Q4: 如何更换图片？

**答案：** 
1. 将图片上传到图床（如腾讯云 COS、Imgur 等）
2. 复制图片 URL
3. 修改数据库中的 `cover_image` 字段

---

#### Q5: 如何添加自定义域名？

**答案：** 
1. 购买域名（如阿里云、腾讯云、GoDaddy）
2. 在云开发控制台绑定域名
3. 配置 DNS 解析（CNAME 记录）
4. 申请 SSL 证书（免费）

---

#### Q6: 费用会超吗？

**答案：** 个人使用完全不会超！

| 服务 | 免费额度 | 个人使用 | 是否够用 |
|------|---------|---------|---------|
| 腾讯云 | 5GB+500 万/月 | <1GB+10 万/月 | ✅ 充足 |
| GitHub Pages | 无限 | - | ✅ 充足 |

---

#### Q7: 如何备份数据？

**答案：** 
- 腾讯云：控制台 → 数据库 → 备份与恢复
- 建议每月备份一次
