💾 数据库架构

数据库集合

集合名说明主要字段
cases 案例主表 id, title, description, cover_image
caseTabs 案例详情 case_id, tab_name, content
visitors 访客记录 visitor_id, ip_address, total_visits
pageViews 页面浏览 visitor_id, page_url, duration
caseViews 案例浏览 case_id, visitor_id, view_duration

📝 命名规范

变量命名

规则：
  • 使用 camelCase（小驼峰）
  • 布尔值用 is/has/are 前缀
  • 常量用 UPPER_SNAKE_CASE
// ✅ 正确 const visitorId = 'xxx'; const isPublished = true; const API_BASE_URL = 'https://...'; // ❌ 错误 const visitor_id = 'xxx'; const published = true;

数据库字段

// 主键：id (integer, auto-increment) // 外键：xxx_id (integer) // 时间：xxx_at (timestamp) // 布尔：is_xxx (boolean) // 计数：xxx_count (integer)
